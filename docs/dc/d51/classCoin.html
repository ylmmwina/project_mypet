Клас Coin

Предмет, який гравець може збирати для отримання очок. Детальніше...

Схема успадкувань для Coin

Загальнодоступні елементи

 constructor (scene, x, y)
 Створює нову монетку.
 interact (player)
 Взаємодія з гравцем.

Детальний опис

Предмет, який гравець може збирати для отримання очок.

  • Монетка має фізичне тіло, стикається з платформами, але не рухається сама по собі (окрім падіння, якщо гравітація увімкнена, хоча тут вона часто статична).

Опис методів компонент

◆ constructor()

Coin::constructor ( scene,
x,
y )
inline

Створює нову монетку.

  • Аргументи
    {Phaser.Scene}scene - Сцена, до якої належить об'єкт.
    {number}x - Координата X.
    {number}y - Координата Y.

◆ interact()

Coin::interact ( player)
inline

Взаємодія з гравцем.

  • Викликається, коли гравець перетинається (overlap) з монеткою. Метод "вимикає" монетку, роблячи її неактивною та невидимою, що імітує її збір.
  • Аргументи
    {Player}player - Об'єкт гравця (не використовується прямо, але передається Phaser'ом).
