Детальний опис

Абстракція предмета, який можна підібрати.

  • Налаштовує об'єкт як нерухомий (immovable) і такий, що не піддається гравітації. Надає метод interact(), який має бути перевизначений у нащадках.

Опис методів компонент

◆ constructor()

Collectable::constructor ( scene,
x,
y,
textureKey )
inline

Конструктор базового збираного об'єкта.

  • Ініціалізує спрайт, додає його до сцени та налаштовує фізичне тіло.
  • Аргументи
    {Phaser.Scene}scene - Поточна сцена гри.
    {number}x - Координата X появи.
    {number}y - Координата Y появи.
    {string}textureKey - Ключ текстури для спрайта.

◆ interact()

Collectable::interact ( player)
inline

Метод взаємодії з гравцем.

  • Викликається при колізії або перетині з гравцем. Базова реалізація просто вимикає об'єкт (збирає його). Цей метод демонструє поліморфізм, оскільки може бути змінений у класах-спадкоємцях.
  • Аргументи
    {Player}player - Екземпляр гравця, що взаємодіє з предметом.
