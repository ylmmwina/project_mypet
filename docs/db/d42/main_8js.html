Детальний опис

Основний лаунчер міні-гри Phaser.

  • Цей файл відповідає за конфігурацію екземпляра гри Phaser.Game, а також надає глобальні функції launchGame та destroyGame для керування грою з основного інтерфейсу (script.js).
