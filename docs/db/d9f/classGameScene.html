Детальний опис

Основна сцена геймплею.

  • Реалізує механіку нескінченного раннера. Світ генерується "чанками" (шматками) по мірі просування гравця вправо. Також відповідає за камеру та очищення об'єктів, що вийшли за межі екрану.

◆ update()

GameScene::update ( )
inline

Ігровий цикл (викликається кожен кадр).

  • Відповідає за:
  • Рух фону.
  • Обробку введення (рух гравця).
  • Поворот спрайта гравця (Flip X).
  • Генерацію нових чанків (нескінченний рівень).
  • Видалення об'єктів, що залишилися позаду (оптимізація).
  • Перевірку падіння у прірву.
