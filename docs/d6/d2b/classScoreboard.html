Детальний опис

Табло рахунку.

  • Клас створює текстовий об'єкт Phaser, закріплює його на екрані (щоб не рухався за камерою) та слухає події гри для оновлення очок.

Scoreboard::addScore ( points = 10)
inline

Додає бали до поточного рахунку.

  • Оновлює числове значення та перемальовує текст на екрані.
  • Аргументи
    {number}[points=10] - Кількість балів для додавання (за замовчуванням 10).
