Детальний опис

Статичний клас для управління сценами.

  • Надає методи для ініціалізації гри, запуску ігрового процесу та обробки завершення гри (Game Over). Дозволяє централізовано керувати потоком гри.

Опис методів компонент

◆ gameOver()

SceneManager::gameOver ( scene,
score )
inlinestatic

Обробка програшу (Game Over).

  • Зупиняє UI, ставить гру на паузу (для ефекту завмирання, щоб гравець бачив, де помилився) та запускає сцену Game Over поверх гри, передаючи їй фінальний рахунок.
  • Аргументи
    {Phaser.Scene}scene - Поточна ігрова сцена (GameScene).
    {number}score - Фінальний рахунок гравця.

◆ startGame()

SceneManager::startGame ( scene)
inlinestatic

Перехід зі стартового меню до гри.

  • Зупиняє меню та сцену Game Over (якщо була), запускає основну гру та UI.
  • Аргументи
    {Phaser.Scene}scene - Поточна активна сцена (зазвичай StartScene або GameOverScene), з якої здійснюється виклик.

◆ startScenes()

SceneManager::startScenes ( game)
inlinestatic

Реєстрація та запуск початкових сцен.

  • Додає всі класи сцен у менеджер сцен Phaser'а та запускає стартове меню. Цей метод викликається лише один раз при ініціалізації гри (main.js або game.js).
  • Аргументи
    {Phaser.Game}game - Екземпляр гри Phaser.
