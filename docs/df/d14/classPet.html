Детальний опис

Клас для представлення віртуального улюбленця.

Зберігає поточні показники стану тварини та надає методи для їх зміни (годування, гра, сон, лікування тощо). Також містить метод live(), який викликається ігровим циклом для симуляції плину часу.

Конструктор класу Pet.

Створює новий екземпляр віртуального улюбленця з заданими параметрами.

@constructor

Аргументи
{string}name - Ім'я улюбленця
{string}type - Тип улюбленця (наприклад, 'cat', 'dog', 'monkey')
{number}age - Вік улюбленця
{number}health - Рівень здоров'я (0-100)
{number}hunger - Рівень голоду (0-100, де 100 - дуже голодний)
{number}happiness - Рівень щастя (0-100)
{number}energy - Рівень енергії (0-100)
{number}cleanliness - Рівень забруднення (0-100, де 100 - дуже брудний)
{number}coins - Кількість монет у власника (за замовчуванням 0)
{number}id - Унікальний ідентифікатор улюбленця в базі даних (за замовчуванням null)
{string}ownerId - Ідентифікатор власника користувача (за замовчуванням null)

◆ live()

Pet::live ( )
inline

Симуляція життєвого циклу (викликається за таймером).

Метод відповідає за природні зміни станів з часом:

  • Збільшується голод і рівень бруду
  • Зменшується щастя
  • Якщо показники критичні (сильний голод, бруд, нещастя), зменшується здоров'я
Нотатки
Здоров'я не може впасти нижче 1 (безсмертя).
