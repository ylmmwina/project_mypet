Детальний опис

Конфігурація товарів та логіка їх впливу.

  • Цей файл містить базу даних доступних для покупки предметів, а також логіку застосування їх ефектів на улюбленців з урахуванням особливостей кожного типу тварини (поліморфізм).

Застосувати ефекти предмета до улюбленця.

  • Ця функція реалізує складну логіку взаємодії предметів з різними типами тварин. Наприклад, мавпи отримують бонуси від бананів, а коти можуть вередувати.
  • Аргументи
    {Object}pet - Об'єкт улюбленця (екземпляр класу Pet).
    {Object}item - Об'єкт предмета з shopItems.

◆ clamp

export const clamp
Початкові значення
= (value, min, max) =>
Math.min(max, Math.max(min, value))

Обмежує числове значення заданим діапазоном.

  • Використовується для того, щоб показники (здоров'я, голод тощо) не виходили за межі 0-100.
  • Аргументи
    {number}value - Вхідне значення.
    {number}min - Мінімальна межа.
    {number}max - Максимальна межа.
    Повертає
    {number} Значення в межах [min, max].
